# 剑指offer 第十三题 
***
## 题目 
地上有一个m行和n列的方格。一个机器人从坐标0,0的格子开始移动，每一次只能向左，右，上，下四个方向移动一格，<br>
但是不能进入行坐标和列坐标的数位之和大于k的格子。 <br>
例如，当k为18时，机器人能够进入方格（35,37），因为3+5+3+7 = 18。<br>
但是，它不能进入方格（35,38），因为3+5+3+8 = 19。请问该机器人能够达到多少个格子？<br>
***
## 思路
BFS 并记录走过的点个数即可

## 额外注意  
输入rows cols小于等于0<br>
threshold小于0 （即初始点也不能到达）
